    -créer la table automatiquement au lancement du serveur si table n'existe pas
    -créer les clé api pour chaque user_id via la relation des table login_schema_db_create_api.login_table_db_create_api et key_api_schema.key_api_table si clé api pour l'user_id n'existe pas
    -lorsque expired_at arrive, on met is_active = fale grâce aux requêtes
