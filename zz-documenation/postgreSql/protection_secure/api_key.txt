------------------------automatisation de la création de clé api-------------------
1-on utilise cron job
2- voici comment ça marche :
    -cron s'active au moment du lancement de serveur 
    -lorsque expired_at arrive, on met is_active fale grâce aux requêtes
    -puis cron vérifie périodiquement la db si is_active est false, il lance la requête de création de clé api et supprime la clé api "is_active = false" de chaque utilisateur
